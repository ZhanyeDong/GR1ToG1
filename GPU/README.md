1、该文件夹是全部计算均使用GPU

2、使用build_sequence_bias_dataset获取高质量GR1机器人的数据集，在train_files处修改输入数据集路径，在output修改输出数据集路径，在bias_abs设定随机范围，在target_samples设置目标数量

3、pinocchio_fk_ik_check是build_sequence_bias_dataset调用的用于清洗的函数，需要在example处更改urdf的文件位置

4、main_G1toGR1是将G1关节映射回GR1的纯由数据训练得到网络

5、goal_datast_build.py是在main.py训练出网络之后，构建从build_sequence_bias_dataset获取的GR1机器人关节角经由main神经网络预测出的G1机器人的关节角度，这个得到的G1机器人的关节角度作为输入值，build_sequence_bias_dataset得到的GR1机器人的关节角度为输出值，用以训练main_G1toGR1
